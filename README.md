## Description

this program get api cypto price and stock with NodeJS Express

## Installation

```bash
$ npm install
```

## Running

```bash
# development
$ npm start

# debug mode
$ npm run dev
```
## How to get 
can import postman.json file to example

## Stock 
http://localhost:3000/stock/price/symbol

example symbol 
- Apple Inc.: AAPL
- Microsoft Corporation: MSFT
- Amazon.com Inc.: AMZN
- Alphabet Inc.: GOOGL


## Cypto Price
http://localhost:3000/cypto/price/symbol

example symbol 
- Bitcoin: BTC
- Ethereum: ETH
- Binance BNB
- Dogecoin: DOGE
